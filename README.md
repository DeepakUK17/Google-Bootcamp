# 🔥 Roast My GitHub

An interactive, visually stunning web application that takes any GitHub username and generates a humorous, data-driven "roast" of their profile and coding habits.

## ✨ Features

- **Live Data Fetching:** Utilizes the GitHub REST API to securely fetch profile metrics, repository data, and language statistics.
- **The "Roast Engine":** A custom logic tier that analyzes followers, account age, repository count, and fork ratios to generate contextual jokes.
- **Dynamic Spice Levels:** Users can select between *Mild*, *Medium*, and *Nuclear* roasts. Selecting *Nuclear* dynamically shifts the entire application's CSS variables to an aggressive red theme.
- **Language Analytics:** Integrates **Chart.js** to render a beautiful, animated donut chart showcasing the user's most frequently used programming languages.
- **Modern UI/UX:** Built with a premium dark mode, glassmorphism CSS effects, and smooth animations.

## 🛠️ Tech Stack

- **Vite:** For lighting-fast scaffolding and hot-module replacement.
- **Vanilla JavaScript:** Proving that you don't always need a heavy framework to build something highly interactive and dynamic.
- **HTML5 / CSS3:** Custom-built styling using CSS variables for theme switching and CSS animations for the glass cards.
- **Chart.js:** For the language distribution analytics.
- **GitHub REST API:** For real-time data sourcing.

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/DeepakUK17/Google-Bootcamp.git
   cd Google-Bootcamp
   ```

2. Install dependencies (Vite):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173/` in your browser and prepare for emotional damage!
